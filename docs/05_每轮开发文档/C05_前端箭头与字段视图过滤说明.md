# C05 补充：前端箭头与字段视图过滤说明

## 本次改了什么

本次只改前端，不改后端代码。

1. 血缘边箭头缩小到原来的 70%。
2. Column 视图只展示字段级血缘节点。
3. 如果后端结果里没有字段级节点，Column 视图返回空图，不再回退展示子查询级或表级血缘。
4. 开发过程中遇到乱码注释时同步清理。

## 字段级血缘当前是否支持

前端支持字段级血缘展示。

支持条件是后端 `graph_view_model.nodes` 里包含字段级节点，例如：

```text
physical_column
output_column
```

当前普通查询和 Join 查询已能返回这类节点时，前端 Column 视图可以展示字段级血缘。

CTE / 子查询结构结果如果暂时只返回 `table / cte / subquery / output`，前端不会把这些结构节点显示在 Column 视图里。
