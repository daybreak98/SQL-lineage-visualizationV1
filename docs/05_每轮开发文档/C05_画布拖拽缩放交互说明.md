# C05 补充：画布拖拽缩放交互说明

## 本次改了什么

本次只改前端画布交互：

1. 支持空白画布拖拽平移。
2. 支持鼠标滚轮缩放。
3. 关闭画布滚动条。
4. 默认缩放会尽量把当前可见血缘图放进一个屏幕内。
5. Reset 会回到自动适配和默认居中状态。

## 交互规则

```text
拖节点：移动单个节点
拖空白画布：平移整张画布
滚轮：围绕鼠标所在位置缩放
Reset：恢复自动适配
```

## 字段级血缘当前状态

前端 Column 视图支持字段级血缘展示。

但它依赖后端返回字段级节点，例如：

```text
physical_column
output_column
```

当前 WITH / CTE 示例 SQL 如果后端只返回：

```text
table
cte
subquery
output
```

那么 Column 视图为空是正常的。前端不会再用结构图冒充字段级血缘图。
